# 数据库约定

1. 数据库编码、表编码、字段编码一律用`utf-8`  
2. 数据库名称、表名称、字段名称、视图名称一律用小写字母  
3. 对于true或false的字段，使用`bit(1)`类型  
4. 对于其他字典类型字段，使用`varchar`类型  
5. 所有表之间使用弱关联关系，不使用强关联关系。级联删除必须由业务代码去实现  
6. 开发中，复杂查询尽量使用视图，不要构建复杂的SQL语句  
7. 所有表必须有ID字段，定义为主键，不能为空  
6. 常用字段名称约定：
   * id
   * name
   * state
   * create_time
   * update_time
   * delete_time
   * expire_time
